# Keyboard shortcuts

Every key in [the TUI](../guides/the-tui.md), by screen. The footer shows the
same keys in context (wrapping to extra rows on narrow terminals).

## Global

Available on every screen:

| Key | Action |
| --- | --- |
| `1` | Dashboard |
| `2` | Timesheet |
| `3` | Clients |
| `4` | Reports |
| `5` | Invoices |
| `6` | Taxes |
| `s` | Start / stop the timer |
| `l` | Quick log |
| `t` | Theme picker |
| `r` | Refresh |
| `q` | Quit |

## Timesheet (`2`)

| Key | Action |
| --- | --- |
| `d` / `w` / `m` | Day / week / month span |
| `[` / `]` | Previous / next period |
| `g` | Jump to today |
| `a` | Add entry |
| `e` | Edit selected entry |
| `x` | Delete selected entry |

## Clients (`3`)

| Key | Action |
| --- | --- |
| `a` | Add client |
| `p` | Add project under selected client |
| `e` | Edit selected client/project |
| `x` | Archive selected |
| `D` | Delete selected |

## Reports (`4`)

| Key | Action |
| --- | --- |
| `w` / `m` | Week / month mode |
| `[` / `]` | Older / newer period |

## Invoices (`5`)

| Key | Action |
| --- | --- |
| `n` | New invoice |
| `o` | Open detail |
| `u` | Refresh (recompute from locked entries) |
| `m` | Preview Markdown |
| `e` | Render PDF + Markdown |
| `t` | Mark sent |
| `p` | Mark paid |
| `v` | Mark void |

## Taxes (`6`)

| Key | Action |
| --- | --- |
| `p` | Record IRS payment |
| `[` / `]` | Previous / next year |

## Modals

| Key | Action |
| --- | --- |
| `enter` | Submit (quick log, forms, pickers) |
| `escape` | Cancel / close |
| `y` | Confirm, in confirmation dialogs |
| `ctrl+s` | Create, in the new-invoice form; apply refresh, in the refresh diff modal |
